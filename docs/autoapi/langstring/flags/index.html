class langstring.flags.GlobalFlag(*args, **kwds)

Bases: enum.Enum

Enumeration for global control flags.

This enum defines various flags that can be used to configure the behavior of all classes.

Variables:

  • DEFINED_TEXT (Enum) – Makes mandatory the use of a non-empty string for the field ‘text’ of all classes.

  • ENSURE_ANY_LANG (Enum) – Makes mandatory the use of a non-empty string for the field ‘lang’ of all classes.

  • VALID_LANG (Enum) – Makes mandatory the use of a valid language code string for all classes’ field ‘lang’.
