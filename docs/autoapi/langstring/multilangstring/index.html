Bases: utils.validation_base.ValidationBase

A class for managing multilingual text strings with various language tags.

Utilizes a global control strategy set in MultiLangStringControl to handle duplicate language tags. Supports operations like adding, removing, and retrieving language strings in multiple languages.

add_langstring(langstring)

Add a LangString to the MultiLangString.

Depending on the current global control strategy (e.g., ALLOW, OVERWRITE, BLOCK_WARN, BLOCK_ERROR), the behavior for handling duplicate language tags varies. For example, BLOCK_ERROR will prevent adding a LangString with a duplicate language tag. For ALLOW, it adds the LangString unless an identical one exists for the same language tag.

Parameters:

langstring (LangString) – The LangString object to be added, representing a text in a specific language.

Return type:

None

get_langstring(lang)

Get LangStrings for a specific language tag.

Returns a list of LangStrings for the specified language tag. If the specified language tag is not present in the MultiLangString, an empty list is returned.

Example:

mls = MultiLangString() mls.add_langstring(LangString(“Hello”, “en”)) print(mls.get_langstring(“en”)) # Output: [“Hello”]

Parameters:

lang (str) – The language tag to retrieve LangStrings for.

Returns:

List of LangStrings for the specified language tag. Returns an empty list if not found.

Return type:

list[str]

remove_language(language_code)

Remove all LangStrings associated with a specific language code from the MultiLangString.

Attempts to remove all LangStrings that match a given language code. Returns True if the language code is found and the entries are successfully removed. Returns False if the language code is not found. Raises a ValueError for invalid language_code formats.

Parameters:

language_code (str) – The language code (e.g., “en”, “fr”) for which to remove LangStrings.

Returns:

True if language entries were removed, False otherwise.

Return type:

bool

Raises:

  • TypeError – If the provided language_code is not a string or is invalid.

  • ValueError – If the provided language_code contains non-alphabetical characters.
